# Templates

Copy-paste starter content for common file types. Always adjust the placeholders, never commit them as-is.

| Template                                | Use for                                   |
| --------------------------------------- | ----------------------------------------- |
| [`feature-spec.md`](feature-spec.md)    | New feature doc in `docs/features/`       |
| [`adr.md`](adr.md)                      | Architecture Decision Records             |
| [`api-endpoint.md`](api-endpoint.md)    | New API endpoint doc                      |
| [`migration.md`](migration.md)          | Alembic migration scaffolding             |
| [`react-component.tsx`](react-component.tsx) | New React component                  |
| [`react-page.tsx`](react-page.tsx)      | New Next.js App Router page               |
| [`react-form.tsx`](react-form.tsx)      | RHF + Zod form                            |
| [`fastapi-endpoint.py`](fastapi-endpoint.py) | New FastAPI endpoint module          |
| [`fastapi-service.py`](fastapi-service.py)   | New service layer module             |
| [`fastapi-repository.py`](fastapi-repository.py) | New repository module             |
| [`sqlalchemy-model.py`](sqlalchemy-model.py)| New ORM model                         |
| [`celery-task.py`](celery-task.py)      | New Celery task                           |
| [`pytest-test.py`](pytest-test.py)      | New pytest test module                    |
| [`playwright-spec.ts`](playwright-spec.ts) | New Playwright spec                    |
| [`log-entry.md`](log-entry.md)          | New entry for any `logs/*.md` file        |
| [`pr-description.md`](pr-description.md)| Pull request body                         |
| [`bug-report.md`](bug-report.md)        | Bug entry for `logs/bug-tracker.md`       |
